5 HTML tags

Heading: <h1></h1>,<h2></h2> <h3></h3>
These tags are used for the heading of a page.
Paragraph: <p></p>
This tag is used when adding a paragraph of text in the page.
Quote: <q></q>
This tag is used to add quotations in the content.
Emphasis: <em></em>
This tag is used to emphasise slightly important text.
Important text: <strong></strong>
This tag is used to emphasise much more important text.
